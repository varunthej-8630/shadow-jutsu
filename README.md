## âš™ï¸ Customization

### Change the confidence threshold

In `script.js`, find:
```javascript

```
Lower it if clones aren't triggering. Raise it if they trigger too easily.

### Change clone positions & sizes

In `script.js`, find the `customClones` array:
```javascript
const customClones = [
  { x: -100, y: 100, scale: 0.9, delay: 1000, smokeSpawned: false },
  // ...
];
```
- `x` â€” horizontal offset from center (negative = left, positive = right)
- `y` â€” vertical offset
- `scale` â€” size of the clone (1.0 = full size, 0.4 = tiny background clone)
- `delay` â€” milliseconds before this clone appears (stagger the spawning!)

### Change colors
